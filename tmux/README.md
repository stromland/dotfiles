# Tmux

## Require

Install dependencies in [Brewfile](../homebrew/Brewfile)

### Popup

- Htop
- Lazygit
- Ranger
  - Highlight - For highlighting files in preview
  - jq - For Json preview

## Keybindings

| Key   | Operation            |
| ----- | -------------------- |
| `C-k` | Prefix               |
| `r`   | Reload tmux config   |
| `C-h` | Window: Previous     |
| `C-l` | Window: Next         |
| `C-q` | Window: Kill         |
| `h`   | Pane: Left           |
| `j`   | Pane: Down           |
| `k`   | Pane: Up             |
| `l`   | Pane: Right          |
| `H`   | Pane: Resize Left 2  |
| `J`   | Pane: Resize Down 2  |
| `K`   | Pane: Resize Up 2    |
| `L`   | Pane: Resize Right 2 |
| `G`   | Popup: Lazygit       |
| `F`   | Popup: Ranger        |
| `S`   | Popup: Htop          |
| `Esc` | Enter copy mode      |
